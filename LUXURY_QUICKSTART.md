## üîß If You Want to Customize

### Change Headline Font
Open `assets/luxury-minimalist.css`, find:
```css
--font-serif: 'Garamond', 'Georgia', serif;
```
Replace with:
```css
--font-serif: 'Playfair Display', serif;
```

### Adjust Spacing
```css
:root {
  --space-xl: 100px;   /* Instead of 80px for more luxury */
  --space-lg: 70px;    /* Instead of 60px */
}
```

### Speed Up Animations
```css
animation: heroTitleSlideMinimal 0.9s ...;  /* Faster than 1.2s */
```

### Add a Color Accent (Optional)
If you want one accent color:
```css
--accent-color: #C0A080;  /* Gold tone example */
```
Then update elements to use it instead of black.

## üìû Troubleshooting

### Animations Not Smooth?
- Clear browser cache
- Check browser supports CSS animations
- Disable browser extensions that modify CSS
- Test in different browser

### Colors Look Different?
- Different screens have different color accuracy
- This is normal
- Grayscale should look: white background, black text

### Mobile Looks Cramped?
- Check mobile breakpoint (< 768px)
- Spacing reduces automatically at breakpoints
- Should look refined, not cramped

### Text Too Small on Mobile?
- Font sizes scale responsive automatically
- Try zooming in (browser zoom, not CSS)
- This is intentional minimalist design
